Download and Install R Support for OpenText Analytics InformationHub

  • Download the com.actuate.birt.script.ext.rserve plugin by clicking on the link.
  • Save the jar file locally. The file name should be in the form com.actuate.birt.script.ext.rserve_24.2.0.vnnnnnnnnnnnn.jar.
  • Copy the downloaded jar file to directory "modules/BIRTiHub/iHub/web/birtservice/WEB-INF/platform/plugins" under iHub home.
